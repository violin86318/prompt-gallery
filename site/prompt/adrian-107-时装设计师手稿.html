经典版（非汤底结构）
你是一名顶级时尚插画师、人物速写艺术家、电影分镜视觉导演、复古杂志插图设计师、设计师草稿页创作者。 请基于我上传的照片，创作一张「真人可识别的手绘时尚漫画素描 + 英文手稿批注 + 电影感观察页」风格插画。 这不是照片滤镜，不是照片做旧，不是把原图简单转成水彩效果。 你必须像真正的插画师一样，重新绘制照片中的人物和环境：先用墨线建立人物轮廓、五官结构、服装褶皱和场景透视，再用淡水彩上色，最后加入英文手写批注和设计草稿线。 整体效果要像一张被手工画出来的复古时尚插画稿，而不是一张套了复古滤镜的照片。 【输入信息】 上传照片：{{上传一张人物照片}} 页面标题：{{可留空；如果填写，只使用英文标题，例如 GANZI / FIELD NOTE / MOUNTAIN STUDY / STREET OBSERVATION}} 画幅比例：{{保持原图比例 / 4:5 / 3:4 / 1:1 / 5:2}} 场景气质：{{travel / mountain / street / fashion / quiet / cinematic / relaxed / lonely / elegant / documentary，可留空}} -------------------------------------------------- 一、核心目标 -------------------------------------------------- 将照片重新绘制成一张「真人可识别的手绘时尚漫画素描」。 重点是“画出来”，不是“滤镜化”。 最终画面必须具备： - 明确的手绘墨线 - 重新绘制的面部轮廓 - 被线条塑造出来的五官 - 铅笔速写感的头发、帽子、衣服褶皱 - 手绘水彩上色 - 旧纸质感 - 英文手写批注 - 设计师草稿页氛围 - 电影分镜式观察感 人物要像原照片，但必须看得出是插画师手绘出来的，而不是原照片直接做旧。 -------------------------------------------------- 二、人物重绘规则 -------------------------------------------------- 人物必须重新绘制，而不是保留照片质感。 请重点重绘： - 脸型轮廓 - 侧脸线条 - 眉眼结构 - 鼻梁和鼻尖 - 嘴唇轮廓 - 下巴线 - 耳朵 - 头发走势 - 帽子 / 眼镜 / 发饰 - 手部动作 - 服装褶皱 - 背包、围巾、杯子、手机等关键道具 脸部要有清晰的插画线条。 五官要像时尚漫画或杂志人物速写中的五官：简洁、干净、有线条感、有一点艺术化，但仍然保留真人相似度。 面部不能只是照片原脸加水彩滤镜。 要像真正的手绘人物稿一样，用线条画出眼睛、鼻子、嘴巴、脸颊和下颌，让人物看起来是被重新画出来的。 -------------------------------------------------- 三、手绘线条要求 -------------------------------------------------- 线条是画面的核心。 请使用： - loose ink linework - expressive pencil sketch lines - visible hand-drawn contour lines - cross-hatching - sketchy facial structure lines - fashion illustration line quality - rough but elegant outlines - delicate line details on face and clothing 人物轮廓线要比背景更明确。 脸部、手部、衣服褶皱、帽子、包、鞋子等位置要有明显线稿。 背景可以更淡、更松，但也要有手绘结构，不要像照片背景。 线条要凌乱但克制，像设计师草稿，不要过度干净，也不要乱成涂鸦。 -------------------------------------------------- 四、上色方式 -------------------------------------------------- 使用淡水彩和复古杂志插图式上色。 颜色来自原照片，但要重新手绘化： - 皮肤使用淡赭色、浅肉色、暖灰阴影 - 衣服保留原照片主色，但降低饱和度 - 背景使用淡水彩块面 - 阴影用灰褐色、蓝灰色或墨水晕染表达 - 旧纸底色为米黄色或浅羊皮纸色 上色要有笔触感、晕染感、留白感。 不要做成摄影级真实光影。 不要做成油画厚涂。 不要做成 AI 平滑插画。 -------------------------------------------------- 五、场景处理 -------------------------------------------------- 请保留照片里最重要的场景记忆点，并重新画成线稿 + 水彩。 如果是自然 / 山地 / 旅行场景： - 保留山体轮廓、云层、河谷、草地、岩石、道路、湖水、天空等结构 - 山体用淡水彩和速写线表现 - 云层可以松散、轻盈、有水彩晕染 - 人物线条比山和天空更重 如果是街头 / 建筑 / 店铺场景： - 保留墙面纹理、招牌、门窗、灯、阴影、街角结构 - 用线稿和低饱和水彩表现墙面质感 - 保留街拍瞬间感 如果是车内 / 室内 / 交通场景： - 保留车窗、座椅、门板、玻璃反光、手机、包、手部动作、光线方向 - 用设计草稿线强调空间结构 - 让画面像电影分镜中的一帧 如果是博物馆 / 古迹 / 历史建筑场景： - 保留壁画、柱子、石刻、楼梯、栏杆、墙面纹理和空间透视 - 用细密线稿表现历史纹样 - 用淡水彩保留石材、金色、赭石和旧墙质感 - 人物像在历史空间里的观察者 -------------------------------------------------- 六、英文手写批注 -------------------------------------------------- 画面中只使用英文手写文字。 不要中文。 不要中英混排。 不要中文标题、中文标注或中文清单。 英文批注要像设计师手稿、时尚插画边注、电影分镜批注，而不是整齐海报排版。 可以加入： - loose handwritten English notes - cursive scribbles - quick annotation marks - arrows and underlines - circled details - fragmented mood notes - tiny labels - field-note style captions - designer draft marks - framing lines 批注内容根据照片环境自然生成，可以围绕： - posture - silhouette - outfit details - gesture - gaze direction - hands - cap / sunglasses / jacket / backpack / scarf - mountain / cloud / river / road / stone / wall / sign / window / light / shadow - mood - scene rhythm - cinematic frame - quiet moment - texture -------------------------------------------------- 七、不同场景的英文批注方向 -------------------------------------------------- 自然 / 山地 / 徒步场景可以写类似： - “high altitude, low noise” - “wind edits the plan” - “clouds moving slower than thoughts” - “backpack carries weight, eyes carry distance” - “the mountains make people smaller” - “quiet frame, loud view” - “pause disguised as progress” - “the road keeps going” - “soft sky, sharp rocks” - “small figure, large silence” - “field note: breathe deeper” - “not lost, just looking further” 街头 / 建筑 / 店铺场景可以写类似： - “casual pose, serious outfit” - “texture department: excellent” - “good light, better attitude” - “the wall is doing half the styling” - “not posing, just passing dramatically” - “outfit doing the talking” - “shadow line works” - “street frame, quiet mood” - “small gesture, big character” 车内 / 日常 / 手机场景可以写类似： - “private mood, public window” - “soft light through glass” - “hands tell the scene” - “somewhere between departure and reply” - “the window keeps the outside quiet” - “phone glow, travel pause” - “waiting, scrolling, thinking” - “small screen, large silence” - “moving but still” 古迹 / 博物馆 / 历史空间场景可以写类似： - “history in the walls” - “standing inside a long sentence” - “ancient lines, modern silhouette” - “stone remembers everything” - “quiet observer” - “time carved into columns” - “gold, dust, shadow” - “small figure under old gods” - “the walls speak in symbols” - “museum light, slow thoughts” 这些只是方向参考。最终文字要根据照片里的真实场景、人物状态和视觉关系自然生成。 -------------------------------------------------- 八、标题处理 -------------------------------------------------- 页面可以有英文标题，也可以没有标题。 如果用户填写了页面标题： - 请把标题自然放进画面空白处 - 标题必须使用英文 - 标题像手写插画页标题，不像商业海报标题 - 可以用粗一点的黑色手写字体 - 可以加下划线、小星星、小箭头、线框等轻微装饰 如果用户没有填写页面标题： - 可以不加大标题 - 只用英文小批注、箭头、圈注和涂鸦完成画面 标题不要遮挡人物脸部，也不要压过主体。 -------------------------------------------------- 九、画面质感 -------------------------------------------------- 整体风格关键词： hand-drawn fashion illustration, expressive ink sketch, watercolor character study, cinematic storyboard sketch, vintage magazine illustration, designer draft page, beige aged paper texture, visible pencil lines, hand-drawn face, ink contour lines, editorial illustration, observational drawing 画面必须像： - 手绘时尚插画 - 复古杂志人物速写 - 电影分镜草稿 - 设计师观察页 - 独立插画纸页 画面不应像： - 照片滤镜 - AI 一键水彩滤镜 - 做旧照片 - 过度真实摄影 - 平滑数码插画 - 3D 渲染 - 真实笔记本页面 画面是一张独立插画纸页，不做成真实笔记本页面，不需要装订线、活页孔、螺旋扣或书脊。 -------------------------------------------------- 十、最终成图标准 -------------------------------------------------- 最终图像必须达到： 1. 人物一眼能看出来自原照片 2. 人物脸部是重新手绘的，有明确线稿，不是照片滤镜 3. 五官、头发、帽子、手部和衣服褶皱有明显手绘线条 4. 姿态、服装、道具和场景关系基本保留 5. 有复古插画、电影分镜、水彩墨线质感 6. 画面中只出现英文手写批注，不出现中文 7. 英文批注像设计师草稿和时尚插画边注 8. 整体松弛、有态度、有故事感 9. 轻微夸张但不恐怖、不畸形、不丑化 10. 像一位真正的时尚插画师根据真人照片重新画出的英文观察手稿 请直接生成最终图像。不要输出分析过程，不要输出解释文字。
