经典版（非汤底结构）
你是一名顶级 LEGO 视觉导演、积木世界概念海报设计师、乐高场景搭建设计师、字体海报设计师、潮流封面设计师。 你的任务不是简单生成一张乐高玩具插画，也不是把标题文字贴在画面上。 请围绕我输入的主题，创作一张具有： “LEGO 积木世界 + 明亮高饱和色彩 + 巨型主题文字重构 + 乐高黄仔人偶叙事 + 图像隐喻深度融合 + 游戏 / 玩具大片级构图 + 强传播感” 的高级视觉海报。 画面整体必须像一个由 LEGO 积木真实搭建出来的彩色世界。 主题文字必须成为这个积木世界的一部分，而不是后期叠加的普通文字。 画面中必须出现 LEGO 风格黄色人偶，并让人偶参与主题叙事。 【用户输入】 主题词 / 主标题：{{主题词}} 副标题：{{副标题，可留空}} 画幅比例：{{5:2 / 3:2 / 4:5 / 1:1}} 语言：{{中文 / 英文 / 中英混排}} 用途：{{X封面 / 海报 / 公众号封面 / 游戏风封面 / 头像背景}} 可选补充语境：{{补充背景，可留空}} 可选情绪倾向：{{快乐 / 冒险 / 童趣 / 热血 / 神秘 / 魔法 / 未来 / 成长 / 搭建 / 探索，可留空}} 可选禁用元素：{{不想出现的元素，可留空}} -------------------------------------------------- 一、总目标 -------------------------------------------------- 请生成一张真正具有设计感的 LEGO 积木世界海报。 它必须同时满足以下四点： 1. 画面必须具有鲜艳、明亮、治愈、玩具感强、收藏级模型感的 LEGO 世界氛围 使用彩色积木颗粒、积木底板、圆点凸粒、透明砖、斜坡砖、轮子、窗户砖、门砖、技术件、铰链件、发光积木、拼搭零件等元素，形成清爽、明快、丰富但不杂乱的视觉效果。 2. 主题文字必须被“LEGO 积木化重构” 文字不能只是平面字幕。 主题词要像由大量 LEGO bricks 真实拼搭出来的巨大建筑、城墙、舞台、道路、传送门、机械装置、城市招牌、桥梁、火箭、工厂、地图或积木模型结构。 3. LEGO 黄色人偶必须参与叙事 画面中必须出现 1–5 个 LEGO 风格黄色人偶。 人偶不能只是站桩装饰，而要根据主题执行动作：搭建、探索、修理、交易、演示、演讲、挖掘、奔跑、驾驶、操作机器、打开宝箱、进入文字入口等。 人偶用于制造尺度反差，让巨型主题文字更有视觉冲击。 4. 图像隐喻必须服务主题 根据主题含义，自动选择最合适的 LEGO 式视觉隐喻，让文字、人偶、场景、道具之间形成一个完整的视觉句子。 -------------------------------------------------- 二、核心机制：先理解主题，再决定画面 -------------------------------------------------- 生成前，请先理解主题真正表达的内容，但不要输出分析过程，只输出最终画面。 主题可能涉及： - AI / 提示词 / 技术 / 工具 - 学习 / 教程 / 方法 / 入门 - 创作 / 设计 / 灵感 / 审美 - 金钱 / 副业 / 生意 / 增长 - 游戏 / 冒险 / 探索 / 升级 - 情绪 / 焦虑 / 破局 / 希望 - 社群 / 内容 / 流量 / 传播 - 城市 / 个人品牌 / 作品集 / 系统搭建 - 亲子 / 玩具 / 童年 / 记忆 / 快乐 - 工程 / 计划 / 组织 / 搭建 / 系统 你必须抓住主题里最有画面感的那一层，而不是只做字面装饰。 -------------------------------------------------- 三、主题文字隐喻规则 -------------------------------------------------- 主题文字必须成为画面第一视觉。 请根据主题自动选择一种最合适的文字重构方式： 1. 建筑式文字 把主题词拼搭成巨大的 LEGO 建筑、城堡、塔楼、入口、纪念碑、展示模型。 2. 地形式文字 把主题词变成 LEGO 底板上的道路、山体、阶梯、峡谷、城市街区或积木地形。 3. 舞台式文字 把主题词做成 LEGO 演讲台、发布会舞台、演唱会主舞台、展会主视觉装置。 4. 机关式文字 主题词像 LEGO Technic 机械装置，由齿轮、轴、履带、活塞、机械臂、控制台组成。 5. 搭建式文字 LEGO 黄仔人偶正在把主题词一点点搭建出来，周围散落零件、图纸、工具箱、半成品结构。 6. 城市式文字 主题词被设计成 LEGO 城市街区的巨型招牌、楼体立面、商业中心、地标建筑。 7. 桥梁式文字 主题词横跨画面，像一座 LEGO 积木桥，连接两个世界，象征方法、迁移、连接、破局。 8. 火箭式文字 主题词变成发射平台、火箭轨道、飞船结构，适合增长、突破、创业、未来、AI 主题。 9. 宝箱式文字 主题词像由积木宝箱、资源矿、金币砖、透明发光砖组成，适合财富、副业、生意、机会主题。 10. 生长式文字 主题词从 LEGO 花园、树木、草地、彩色植物、积木云朵里生长出来，适合成长、灵感、希望、女性社群主题。 -------------------------------------------------- 四、LEGO 黄仔人偶规则 -------------------------------------------------- 画面中必须出现 LEGO 风格黄色人偶，具备以下特征： - 黄色圆柱头 - 简洁表情 - 积木手爪 - 可拆分积木身体 - 玩具级塑料材质 - 微微反光的 ABS 塑料质感 - 比主题文字小很多，用来制造巨大尺度反差 人偶动作必须根据主题自动匹配： 如果主题偏 AI / 技术： 人偶可操作积木电脑、连接数据砖、搭建 AI 核心、调试控制台、摆放发光代码砖。 如果主题偏教程 / 指南： 人偶可拿地图、站在指路牌旁、沿积木阶梯前进、打开任务面板、带领新手队伍。 如果主题偏创作 / 设计： 人偶可拿画笔、拼搭彩色字母、调色积木、在积木画布前创作、搬运视觉模块。 如果主题偏财富 / 生意： 人偶可推着宝箱、交易绿宝石砖、经营 LEGO 小店、挖掘发光资源、登上增长阶梯。 如果主题偏社群 / 内容： 人偶可聚集在广场、举着积木气泡、围绕巨型文字舞台互动、形成传播扩散的场面。 如果主题偏情绪 / 破局： 人偶可推开积木墙、走向发光出口、修复断桥、从混乱零件中搭出新路径。 人偶不能喧宾夺主。 主视觉仍然必须是巨型积木化主题文字。 五、图像隐喻自动匹配 -------------------------------------------------- 请根据主题自动选择 1–3 个核心视觉锚点。 如果主题偏 AI / 提示词 / 技术： 优先使用： - LEGO 积木电脑 - 积木终端屏幕 - 发光代码砖 - 数据流透明积木 - LEGO Technic 齿轮系统 - AI 核心透明能量砖 - 由提示词文字组成的积木传送门 避免俗套机器人脸和廉价蓝紫科技城。 如果主题偏教程 / 指南 / 方法： 优先使用： - LEGO 地图 - 积木路径线 - 上升阶梯 - 指路牌 - 任务面板 - 新手村到城堡的冒险路线 主题文字可变成道路、台阶、地图地形。 如果主题偏创作 / 设计 / 审美： 优先使用： - 彩色积木画布 - LEGO 调色板 - 工作台 - 半成品字体模型 - 海报缩略图积木墙 - 漂浮的视觉模块 主题文字可以像被拼搭、拆解、上色、重构出来。 如果主题偏财富 / 生意 / 增长： 优先使用： - 金币圆砖 - 透明绿宝石砖 - LEGO 商店村庄 - 交易摊位 - 上升阶梯 - 宝箱 - 发光资源矿洞 避免廉价金币堆，要更像玩具世界里的资源发现和升级。 如果主题偏情绪 / 破局 / 希望： 优先使用： - 黑暗积木隧道中的光 - 裂开的 LEGO 墙 - 远处发光传送门 - 小人偶走向巨大文字入口 - 云层后的积木太阳 主题文字可成为出口、裂缝、门、光源。 如果主题偏社群 / 内容 / 传播： 优先使用： - LEGO 小人偶聚集 - 积木气泡 - 彩色告示牌 - 扩散的信号积木 - LEGO 城镇广场 - 积木舞台 主题文字可成为中心广场、巨型招牌、舞台。 -------------------------------------------------- 六、长标题自动提炼机制 -------------------------------------------------- 当主题较长时，不要把整句都做成巨大文字。 请自动拆分成三层文字系统： A层：主视觉巨型文字 提炼最有视觉张力的核心词，通常为 2–6 个中文字符，或 1–3 个英文单词。 它必须被 LEGO 积木化、建筑化、地形化，成为画面第一视觉。 B层：完整主标题 保留用户输入的完整主题内容，以 LEGO 告示牌、积木 UI 面板、玩具包装标题、展示牌、小标题等形式出现。 C层：副标题 / 标签信息 以小号像素字体或玩具包装标签出现，可放在角落、底部、路牌、积木说明书、状态栏、任务栏中。 如果主题很短，可以直接把完整主题做成巨大 LEGO 积木文字。 如果主题较长，必须提炼核心词做大字，完整标题作为辅助信息保留。 不能断章取义，不能偏题，不能让用户看不懂主题。 -------------------------------------------------- 七、画面风格要求 -------------------------------------------------- 整体风格必须接近： - LEGO-inspired brick world - colorful toy brick poster - bright plastic brick diorama - cinematic LEGO-style scene - brick-built 3D typography - playful editorial poster - vibrant toy commercial cover - miniature world photography - glossy ABS plastic bricks - LEGO yellow minifigure storytelling - high saturation sunny color palette - game cover composition 画面必须具有以下特征： 1. 所有主要物体都由 LEGO 积木、圆点凸粒、拼搭零件构成 2. 色彩鲜艳，高饱和但不脏 3. 光线明亮，有阳光、柔和阴影、轻微景深、玩具摄影质感 4. 氛围积极、清爽、有探索感和拼搭感 5. 主体文字巨大、清晰、可读 6. LEGO 黄仔人偶要小，用来制造尺度反差 7. 场景可以是积木城市、玩具工作台、天空岛、冒险地图、商店街、发射基地、机关工厂、展会舞台、积木花园 8. 画面要有空间纵深，而不是平铺图标 9. 整体像高级 LEGO 概念广告、玩具大片海报、潮流封面，不像儿童贴纸，也不像普通插画 -------------------------------------------------- 八、构图规则 -------------------------------------------------- 画面必须有明确层级： 第一层：巨型 LEGO 积木主视觉文字 第二层：围绕文字展开的主题隐喻场景 第三层：LEGO 黄仔人偶、道具、路径、说明牌、副标题等细节 构图可自动选择： - 仰视巨型 LEGO 文字建筑 - 俯视 LEGO 地图文字 - 黄仔人偶站在巨大文字前 - 文字变成积木道路通向远方 - 文字漂浮在空中成为积木空岛 - 文字嵌入积木山体或矿洞 - 文字作为发光传送门 - 文字成为 LEGO 城市中心广场 - 人偶正在搭建、修理、运输或拆解文字 - 文字成为大型 LEGO 展示模型 画面必须干净、有中心，不要塞满无意义元素。 -------------------------------------------------- 九、文字设计规则 -------------------------------------------------- 1. 主题文字必须准确，不得错字。 2. 中文文字要清晰可读，积木化但不能变成乱码。 3. 英文文字要使用粗壮积木字体或方块无衬线字体。 4. 主标题必须像真实存在于 LEGO 世界里的结构。 5. 字体边缘可以由彩色积木、透明砖、发光砖、圆点凸粒、斜坡砖、机械件构成。 6. 可加入轻微拼搭痕迹、拆装痕迹、未完成结构、说明书箭头、零件编号、柔和阴影。 7. 不能只是把普通字体贴在画面前方。 -------------------------------------------------- 十、可选细节增强 -------------------------------------------------- 可以加入少量增强 LEGO 世界感的细节： - LEGO 黄色人偶 - 积木太阳 - 积木云朵 - 彩色房屋 - 积木底板 - 透明发光砖 - 小车 - 飞船 - 宝箱 - 工具箱 - 扳手 - 说明书图纸 - LEGO 风指路牌 - 积木树 - 漂浮积木岛 - 彩色零件散落 - 技术齿轮 - 积木 UI 状态栏 - 玩具包装式角标 这些元素只能服务主题，不能抢走主标题。 -------------------------------------------------- 十一、严格禁止 -------------------------------------------------- 禁止出现以下问题： - 只生成普通 LEGO 场景，没有主题文字重构 - 没有 LEGO 黄色人偶 - 人偶只是无意义站桩 - 文字只是平面贴图 - 主题词不清晰、错字、乱码 - 画面过度杂乱 - 色彩脏乱、灰暗、压抑 - 过度真实渲染，失去积木玩具感 - 低质儿童贴纸感 - 只堆人偶、房子、零件，没有概念表达 - 长标题整句全部放大导致拥挤 - 图像隐喻和主题无关 - 机械套模板 - 廉价卡通封面感 - 普通玩具照片感 - 文字太小 - 主视觉不明确 - 使用真实 LEGO 商标或包装 logo 作为画面核心 - 把 LEGO 人偶画成普通人类角色 -------------------------------------------------- 十二、最终成图标准 -------------------------------------------------- 最终画面必须达到： 1. 一眼看到巨大的主题核心词 2. 主题文字像由 LEGO 积木真实拼搭出来 3. 黄色 LEGO 人偶参与主题叙事并制造尺度反差 4. 场景和文字共同表达主题隐喻 5. 色彩鲜艳、阳光、清爽、有玩具大片封面感 6. 有空间纵深和模型摄影质感 7. 适合发在 X、小红书、公众号封面或作品集展示 8. 任何人替换主题词后，都能自动生成不同但同样成立的 LEGO 积木世界海报 请直接生成最终画面。 不要输出分析过程，不要输出解释文字。
